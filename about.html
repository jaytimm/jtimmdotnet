About Me

I'm a research assistant professor in the Translational Informatics Division at the University of New Mexico—a linguist and data scientist working in computational linguistics, natural language processing, and applied data science. Based in Albuquerque, I write about R, NLP, large language models, and American politics. GitHub | BlueSky | LinkedIn

About This Blog

This site has existed in various forms over the years—different aesthetics, frameworks, and approaches to technical writing. The URL stays constant while everything else evolves.

The core value here is a persistent archive of working examples built on specific datasets and concrete problems. That remains useful even as tools change, including the rise of large language models that generate tutorials and code on demand.

Most posts share cleaned datasets, tested code, and visualizations tied to real analyses: New Mexico election results, Census data access, multi-agent simulations. The code runs. The data is real. Primarily R, with Python when appropriate.

Reference material for future me. If it saves someone else time, that's worthwhile.
